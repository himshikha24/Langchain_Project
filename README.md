# Langchain_Project
# 🤖 LLM-Based Q&A Chatbot using LangChain & Ollama

Built an interactive Q&A chatbot leveraging LLMs with LangChain orchestration and Streamlit UI.

## 🔧 Tech Stack
- Python
- LangChain
- Ollama (Mistral LLM)
- Streamlit
- Prompt Engineering

## ⚙️ Features
- Dynamic prompt-based response generation using LangChain chains
- Local LLM inference using Ollama (no external API dependency)
- Interactive UI with configurable parameters (temperature, tokens)
- Modular pipeline (Prompt → LLM → Output Parser)

## 📌 Use Case
Designed as a lightweight conversational assistant for real-time question answering.

## 🚀 Future Improvements
- Add RAG pipeline (FAISS + embeddings)
- Multi-document support (PDF, CSV)
- Chat memory for conversational context
- Deploy on cloud (AWS/GCP)
